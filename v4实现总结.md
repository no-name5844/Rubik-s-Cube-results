# 魔方比赛管理系统 v4 - 实现总结

## ✅ 已完成的工作

### 1. 数据库层级结构 (schema-v4.sql)

**新增字段：**
- `events.parent_event_id` - 自引用，支持父子项目
- `events.is_sub_event` - 是否为子项目
- `events.sort_order` - 排序权重
- `events.algorithm_config` - 算法配置 (JSONB)
- `events.event_config` - 项目配置 (JSONB)

**新增表：**
- `event_algorithms` - 算法模板库（内置 8 种算法）

**内置算法模板：**
| 代码 | 名称 | 说明 |
|------|------|------|
| `single` | 单次最佳 | 取最佳单次成绩 |
| `average_trim1` | 去头尾平均 | WCA 标准平均 |
| `ao5` | AO5 | 最近 5 次平均（去头尾）|
| `ao12` | AO12 | 最近 12 次平均（去头尾）|
| `ao50` | AO50 | 最近 50 次平均 |
| `ao100` | AO100 | 最近 100 次平均 |
| `best_of_3` | 三局两胜 | 取最佳 3 次中的最佳 |
| `sub_x` | SUB-X | 统计低于某时间的比率 |
| `mean` | 算术平均 | 简单算术平均（不去头尾）|

**数据库函数：**
1. `get_event_full_config(event_id)` - 获取项目完整配置（自动继承父项目）
2. `calculate_statistic(participant_id, event_id)` - 根据算法配置动态计算统计值

async function loadEventsHierarchical() {
    // 1. 查询项目（带父项目信息）
    const { data, error } = await dbClient
        .from('events')
        .select('*, parent:parent_event_id(event_code, event_name)')
        .order('COALESCE(parent_event_id::text, id::text), sort_order, event_code');
    
    if (error) { /* 处理错误 */ }
    
    // 2. 组织层级数据
    const parentEvents = data.filter(e => !e.parent_event_id);
    const childEvents = data.filter(e => e.parent_event_id);
    
    const hierarchicalData = [];
    parentEvents.forEach(parent => {
        hierarchicalData.push({
            ...parent,
            _indent: 0,
            _is_parent: true
        });
        childEvents
            .filter(child => child.parent_event_id === parent.id)
            .forEach(child => {
                hierarchicalData.push({
                    ...child,
                    _indent: 1,
                    parent_code: parent.event_code,
                    parent_name: parent.event_name
                });
            });
    });
    
    // 3. 渲染表格（带缩进）
    if (eventsTable) eventsTable.destroy();
    eventsTable = new Tabulator('#events-table', {
        data: hierarchicalData,
        layout: 'fitColumns',
        columns: [
            {
                title: '项目代码',
                field: 'event_code',
                formatter: (cell) => {
                    const data = cell.getRow().getData();
                    const indent = data._indent || 0;
                    const prefix = indent > 0 ? '  ↳ ' : '';
                    return `<span style="padding-left: ${indent * 20}px;">${prefix}${data.event_code}</span>`;
                }
            },
            { title: '项目名称', field: 'event_name' },
            { 
                title: '父项目', 
                formatter: (cell) => {
                    const data = cell.getRow().getData();
                    return data.parent_name ? `${data.parent_code} - ${data.parent_name}` : '-';
                }
            },
            { 
                title: '算法类型', 
                formatter: (cell) => {
                    const data = cell.getRow().getData();
                    const algo = data.algorithm_config || {};
                    return algo.algorithm_type || '-';
                }
            }
        ]
    });
}

### 2. 前端：使用数据库函数计算统计

**需要修改 `loadStats()` 函数：**

```javascript
async function loadStats() {
    const participantId = document.getElementById('stats-participant').value;
    const eventId = document.getElementById('stats-event').value;
    
    if (!participantId || !eventId) return;
    
    // 使用数据库函数 calculate_statistic() 动态计算
    const { data, error } = await dbClient
        .rpc('calculate_statistic', {
            p_participant_id: participantId,
            p_event_id: eventId
        });
    
    if (error) {
        showAlert('计算失败：' + error.message, 'error');
        return;
    }
    
    // 显示结果
    document.getElementById('stats-result').innerHTML = `
        <div class="stat-card">
            <h3>计算结果</h3>
            <div class="value">${data || '-'}</div>
        </div>
    `;
}
```
